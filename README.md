如果你希望快速简单地解密文件，可以选择独立使用 Unlock.js 脚本。
 
1. 将 `unlock.js` 文件复制到含有你想要解密文件或文件夹的目录下。
2. 打开命令行终端，并切换到该目录。
3. 运行以下命令，将 `path/to/encrypted/file` 替换为你要解密的文件的实际路径：
```sh
node unlock.js path/to/encrypted/file
```
